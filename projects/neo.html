Find Near-Earth Objects (NEO)

This application allows you to search for near-Earth objects on a specific day. It uses the NeoWs (Near Earth Object Web Service) to access the data about all near-Earth objects on the desired day. A GIF of the application can be seen below. You can try it for yourself here.
